title: 如何创建和管理 NimbusFlow API Key
topic: integrations_api
---

要通过 API 访问 NimbusFlow，您需要先创建 API Key。请进入 设置>集成>API，点击「生成新密钥」，为密钥命名并选择权限范围（scope）。密钥生成后只会完整显示一次，请立即复制并安全保存；离开页面后只能看到末尾几位。在 API 请求中通过 Authorization: Bearer <token> 请求头携带密钥。若密钥泄露，请立即在同一页面「吊销」（revoke）该密钥并重新生成。默认 API 速率限制为每分钟 600 次请求，超出会返回 429 状态码。
